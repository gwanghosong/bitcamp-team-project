# UC002 - 팀 정보(MyTeamInfo)

팀을 조회, 탈퇴 하는 유스케이스이다.

## 주 액터(Primary Actor)

팀장, 팀원

## 보조 액터(Secondary Actor)

## 사전 조건(Preconditions)

- 팀장으로 로그인 되어 있다.
- 팀원으로 로그인 되어 있다.

## 종료 조건(Postconditions)

- 팀을 조회하였다.
- 팀을 탈퇴하였다.

## 시나리오(Flow of Evnets)

### 팀 조회하기

1. 엑터는 나의 팀 정보 메뉴를 클린한다.
2. 시스템은 팀정보 폼을 출력한다.
    
### 팀 탈퇴하기

1. 엑터는 팀 탈퇴하기 버튼을 클릭한다.
2. 시스템은 팀 탈퇴 확인 사실을 출력한다.
3. 엑터는 확인 버튼을 클릭한다.
4. 시스템은 팀 정보에서 엑터를 제외한 후 '메인' 유스케이스로 간다.
